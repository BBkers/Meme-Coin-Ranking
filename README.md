
# Solana Meme Coin Intelligence Platform

## Features
- Real-time WebSocket updates
- Redis caching
- Telegram alerts
- Wallet tracking (Helius)
- Tailwind trading UI

## Quick Start

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm run dev

### Env
Create backend/.env:
REDIS_URL=
TELEGRAM_BOT_TOKEN=
TELEGRAM_CHAT_ID=
HELIUS_API_KEY=
